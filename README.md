# QQAssist v1.0.0

**Полноценный ассистент для Minecraft сервера с интеграцией Telegram Mini App**

Автор: AllF1RE | Версия Minecraft: 1.21.x (Paper/Spigot)

---

## 📋 Возможности

### 🤖 Чат-бот
- Настраиваемые правила ответов с поддержкой точных фраз, регулярных выражений и поиска по словам
- Случайные ответы, кулдауны, приоритеты, шансы срабатывания
- Условия через PlaceholderAPI, проверка прав, задержки ответов
- Выполнение любых команд от игрока или консоли

### 📱 Telegram Mini App
- Встроенный HTTP-сервер для Mini App (HTML/CSS/JS)
- Полностью настраиваемый интерфейс через конфиги
- Поддержка светлой и тёмной темы

### 🎁 Система очков и Check-in
- Ежедневная отметка с наградой (1-30 очков)
- Серия отметок (streak)
- Передача очков между игроками
- Место в топе по очкам

### 🛒 Магазин
- Покупка внутриигровых предметов и услуг за очки
- Категории товаров, кулдауны, кастомные команды
- Полная настройка через shop.yml

### 🚩 Система жалоб
- Жалобы из игры (/report) и из Mini App
- Просмотр и управление жалобами для администраторов
- Наказания (warn, mute, ban, jail) через Mini App
- Награждение отправителей подтверждённых жалоб

### 🔗 Привязка Telegram
- Привязка Minecraft аккаунта к Telegram через Mini App
- Кодовая система подтверждения
- PlaceholderAPI плейсхолдеры

## 📦 Установка

1. Скачай `QQAssist-1.0.0.jar` и помести в папку `plugins/`
2. Перезапусти сервер (или загрузи плагин через PlugMan)
3. Настрой конфигурационные файлы в `plugins/QQAssist/`
4. Выполни `/qqassist reload` для применения изменений

### Зависимости
- **Paper/Spigot 1.21.x** (обязательно)
- **PlaceholderAPI** (опционально, для условий и плейсхолдеров)
- **CMI** (опционально, для команд магазина)
- **PlayerPoints** (опционально, для отображения в профиле)
- **LuckPerms** (опционально, для команд магазина)

## 🎮 Команды

### Для игроков
| Команда | Алиасы | Описание |
|---------|--------|----------|
| `/qqassist` | `/qq`, `/qqa` | Основная команда плагина |
| `/qqassist help` | - | Показать справку |
| `/qqassist info` | - | Информация о плагине |
| `/checkin` | `/ci`, `/daily` | Ежедневная отметка |
| `/qqpoints` | `/qqp`, `/points` | Проверить свои очки |
| `/qqlink <код>` | - | Привязать Telegram аккаунт |
| `/qqunlink` | - | Отвязать Telegram аккаунт |
| `/qqshop` | - | Просмотр магазина |
| `/report <игрок> <категория>` | `/rep`, `/ticket` | Отправить жалобу |

### Для администраторов
| Команда | Описание |
|---------|----------|
| `/qqassist reload` | Перезагрузить конфигурацию |
| `/qqadmin points reset <игрок>` | Сбросить очки игрока |
| `/qqadmin points set <игрок> <N>` | Установить очки |
| `/qqadmin points add <игрок> <N>` | Добавить очки |
| `/qqadmin points take <игрок> <N>` | Забрать очки |
| `/qqadmin reports` | Список жалоб |
| `/qqadmin reports view <id>` | Просмотр жалобы |
| `/qqadmin reports take <id>` | Взять жалобу |
| `/qqadmin reports close <id>` | Закрыть жалобу |
| `/qqadmin punish <тип> <игрок> <время> <причина>` | Наказать игрока |

## 🔧 Права

| Право | По умолчанию | Описание |
|-------|--------------|----------|
| `qqassist.use` | Все | Основные команды |
| `qqassist.report` | Все | Отправка жалоб |
| `qqassist.admin` | OP | Админ-команды |
| `qqassist.admin.reports` | OP | Управление жалобами |
| `qqassist.admin.points` | OP | Управление очками |
| `qqassist.admin.punish` | OP | Наказание игроков |
| `qqassist.bypass.chat` | OP | Игнорирование чат-ботом |
| `qqassist.bypass.checkin_cooldown` | OP | Обход кулдауна check-in |
| `qqassist.vip` | Нет | VIP доступ к товарам |

## ⚙️ Конфигурация

### Основные файлы в `plugins/QQAssist/`

| Файл | Назначение |
|------|------------|
| `config.yml` | Основные настройки, тема Mini App, наказания |
| `lang.yml` | Все сообщения плагина на русском |
| `responses.yml` | Правила чат-бота |
| `shop.yml` | Товары магазина |
| `reports.yml` | Категории жалоб |
